Project Overview

We built a complete Java Chess game with a clean, responsive GUI layered over a tested console engine. The goal: fast, intuitive gameplay with accurate rule enforcement, clear visuals, and a frictionless two-player experience.

Our Approach

We designed the engine and UI as separate systems. Core logic—board state, piece movement, turn flow—was implemented first, then connected to a minimal, responsive GUI. Each phase refined structure, reduced noise, and strengthened reliability.

Key Features

  • Legal-move validation

  • Capture logic

  • Check / checkmate detection

  • Turn-based two-player flow

  • Responsive 8×8 GUI with mouse interaction
